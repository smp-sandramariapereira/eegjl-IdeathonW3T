# Project ID Guidelines

Use projectId values that do not identify a student or class.

## Do not use

- name;
- e-mail;
- enrollment number;
- class group;
- date of birth.

## Use instead

- a generic project label;
- a short functional code;
- an internal demo identifier that has no direct personal meaning.

## Examples allowed

- `library-project-demo`
- `class-evidence-alpha`
- `proof-pilot-01`

## Examples prohibited

- `maria-silva-2026`
- `student123@email.com`
- `class-3A-joao`
- `2008-04-18`
