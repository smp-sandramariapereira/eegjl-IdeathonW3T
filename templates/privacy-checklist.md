# Privacy Checklist

Before issuing evidence, confirm:

- [ ] projectId is non-personal
- [ ] URI does not expose a name
- [ ] file contains no personal data
- [ ] hash was generated locally
- [ ] wallet is a test wallet
- [ ] network is Sepolia
- [ ] evidence was reviewed
- [ ] no secret is in the repository

If any item is unchecked, stop and revise the record before publication.
