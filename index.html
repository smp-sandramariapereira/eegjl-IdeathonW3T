ProvaOnChain · Minimal Interface

Issue and verify academic evidence on Ethereum.

This page is a minimal browser UI for the early-stage ProvaOnChain pilot. It records only non-personal identifiers, hashes, URIs, issuers, and timestamps. Student identity, grades, and detailed assessments stay offchain.

Session setup

Connect a browser wallet and set the deployed contract address. The same interface works with local Hardhat or Sepolia, as long as the wallet is on the matching network.

Issue evidence

Upload a file or paste the evidence text. The page hashes the content locally with SHA-256 and sends only the hash plus metadata onchain.

Verify evidence

Load a public record by evidence ID, then compare the onchain hash with a local hash computed from the same file or text.
