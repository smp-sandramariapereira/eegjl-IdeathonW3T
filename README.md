# ProvaOnChain

ProvaOnChain is an educational project for registering and verifying cryptographic evidence of academic artifacts, with student data kept offchain and privacy preserved by design.

The Web3 Ideathon methodology is protocol-independent. The first pilot implementation used Solana. An Ethereum implementation of ProvaOnChain is currently being designed and prepared.

## Resumo em Português

ProvaOnChain é uma base documental para um piloto Ethereum voltado à verificação de evidências acadêmicas, com dados de estudantes fora da blockchain e foco em privacidade, segurança e rastreabilidade.

## What this repository is for

This repository documents the methodological basis, the Solana pilot history, and the planned Ethereum pilot for ProvaOnChain. It is a documentation-first repository and does not claim a deployed Ethereum integration.

The planned Ethereum work focuses on public testnet usage, non-personal identifiers, hashes, timestamps, and verifiable evidence records only.

## Core principles

- Protocol choice must answer a technical need, not a trend.
- Academic evidence must be verifiable without exposing student identity.
- Onchain data must stay minimal and non-personal.
- Offchain documents keep the educational context and any sensitive information.
- Public documentation must state status honestly: planned, under design, or in preparation.
