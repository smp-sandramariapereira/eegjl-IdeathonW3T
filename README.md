# Organiza-o-Ideathon-W3T---Escola-Estadual-Geraldo-Jardim-Linhares
A Superteam Brasil patrocina a realização de um Ideathon W3T na Escola Estadual Geral Jardim Linhares, com o objetivo de introduzir estudantes ao ecossistema Web3/Solana de forma prática, incentivando a colaboração, a formação de equipes e a competitividade saudável.
